## Virgool Article Crawler
A crawler for [Hanieh Mahdavi Virgool Account](https://virgool.io/@haniehmahdavi26) to crawl article titles and links.

### Features
+ implemented in Python
+ crawl dynamic data using Selenium

### Setup
```console
python run.py
```
